## 🚀 Quick Installation

### 1. Clone the repository
```bash
git clone https://github.com/verse91/.dotfiles.git ~/.dotfiles
````

### 2. Navigate to the installer directory

```bash
cd ~/.dotfiles/scripts/installer
```

### 3. Run the installation script

```bash
sudo sh install.sh
```

## ⌨️ Key Bindings

**Applications**

* Super + T → Open terminal (\$terminal)
* Super + B → Open browser (\$browser)
* Super + R → Reboot now
* Super + S → Shut down now
* Super + F → Open file manager (\$fileManager)
* Super + A → Open application menu (\$menu)
* Super + M → Exit Hyprland
* Super + N → Set wallpaper (`swww img ~/.config/assets/backgrounds/cat_leaves.png`)

**Window Management & Workspace Navigation**

* Super + Q → Close active window
* Super + W → Toggle floating mode for active window
* Super + J → Toggle split mode in Dwindle layout
* Super + \[Arrow Keys] → Move focus between windows
* Super + Shift + \[Arrow Keys] → Move active window
* Super + Ctrl + \[Arrow Keys] → Resize active window
* Super + \[1-9] → Switch to workspace 1–9
* Super + Shift + \[1-9] → Move active window to workspace 1–9
* Super + Mouse Scroll → Scroll through workspaces
* Alt + Tab / Alt + Q → Switch workspaces

**Screen, Volume & Media Controls**

* Brightness Up → Increase brightness 5%
* Brightness Down → Decrease brightness 5%
* Volume Up → Increase volume 5%
* Volume Down → Decrease volume 5%
* Audio Mute → Mute audio
* Play/Pause → Toggle media
* Next Track → Next media track
* Previous Track → Previous media track

**Clipboard & Utilities**

* Super + V → Open clipboard history and paste
* Super + P → Open color picker and copy color
* Super + L → Lock screen
* Super + Escape → Open logout menu
* Ctrl + Escape → Toggle Waybar
* Print Screen → Screenshot entire screen
* Super + Print Screen → Screenshot active window
* Super + Shift + Print Screen → Select area screenshot
* Super + E → Emoji picker + copy to clipboard

**Mouse Window Control**

* MainMod + LMB → Move window
* MainMod + RMB → Resize window
* MainMod + Z → Move window (alternate)
* MainMod + X → Resize window (alternate)

**Window Resize Shortcuts**

* MainMod + Shift + Right → Increase width 30px
* MainMod + Shift + Left → Decrease width 30px
* MainMod + Shift + Up → Decrease height 30px
* MainMod + Shift + Down → Increase height 30px
